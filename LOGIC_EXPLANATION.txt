# Application Logic - Simple Explanation

## Overview
This tool helps analyze how international financial aid (grants) affects currency values in developing countries, and identifies unusual currency movements.

---

## 1. Grant Impact Analysis

### What We Measure
When a country receives a large grant, we check if the currency became stronger or weaker in the following weeks.

### How We Score Impact (0-100)

**Three Key Factors:**

1. **Trend Change (30% weight)**
   - Did the currency's direction change after the grant?
   - Example: If currency was weakening before, did it start strengthening after?
   - Score: 0 = no change, 100 = complete reversal

2. **Volatility Change (50% weight)**
   - Did the currency become more stable or more unstable?
   - We prefer less volatility (more stability)
   - Score: 100 = much more stable, 0 = much more unstable

3. **Magnitude (20% weight)**
   - How much did the currency value actually change?
   - Bigger changes = higher scores
   - Score based on percentage movement

**Final Score:** Combines all three factors to give a 0-100 rating
- 80-100: Strong positive impact
- 60-79: Moderate impact
- 40-59: Weak impact
- Below 40: Minimal or negative impact

---

## 2. Anomaly Detection

### What We Look For
Unusual currency movements - when a currency changes by more than 15% in less than 30 days.

### Detection Parameters
- **Volatility Threshold:** How big the change must be (e.g., 15%)
- **Time Window:** How quickly it happened (e.g., 30 days)

### Why It Matters
Sharp currency movements often signal economic events that need investigation.

## 4. Data Time Periods

### Default Analysis Window
- **Historical Data:** From 2010 to present
- **Grant Analysis:** Compares 4-12 weeks before/after grant
- **Anomaly Search:** Scans entire dataset

### Why Historical Data?
- Shows patterns over time
- Identifies if current situations are unusual
- Validates relationships between grants and currency

## 5. Confidence & Reliability

### What Makes Results Reliable?
1. **Multiple Data Sources:** 
   - Yahoo Finance (FX rates, commodities)
   - IMF & World Bank (grant data)
   - Economic APIs (policy rates, news)

2. **Outlier Removal:**
   - Removes bad data points (e.g., obvious errors)
   - Uses statistical methods (3-sigma rule)

3. **Caching Strategy:**
   - Stores results to avoid repetitive calculations
   - 12-hour refresh for market data
